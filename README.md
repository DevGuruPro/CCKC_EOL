# CCKC_EOL
Software for writing serial number and manufacturing datestamp from handscanner input.

# Getting Started

- Clone this repository:
    ```shell
    cd ~
    git clone https://github.com/Syntbit/CCKC_EOL
    ```
- Install dependencies

    ```shell
    cd CCKC_EOL
    bash setup.sh
    ```

- And reboot, that's all! :)
